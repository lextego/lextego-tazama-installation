Add this to your docker-compose.yaml:

```yaml
# Demo UI
demoUI:
    image: ruhfuskdev/tazama-demo:v1.0.15
    env_file:
      - path: ./env/demo.env
        required: true
    restart: always
    depends_on:
      - tms
      - arango
      - nats
    ports:
      - '3001:3001'
```

We now need to create a ``demo.env`` file in the ```full-stack-docker-tazama/env``` folder with the contents:
